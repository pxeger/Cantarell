# Cantarell
Mirror/conversions of Cantarell; specifically version `0.301`.

- [the Arch Linux repository](https://archlinux.org/packages/extra/any/cantarell-fonts)
- [GNOME download page](https://download-fallback.gnome.org/sources/cantarell-fonts/0.301/)
- [Source code](https://gitlab.gnome.org/GNOME/cantarell-fonts/)
- [Converter I used](https://www.fontsquirrel.com/tools/webfont-generator)

Note: The version of Cantarell available on Google Fonts and in various other places seems to be outdated, because it's
a bit different from this version.
